# 配置env

将 .env.example  文件修改为 .env

#PROXYUSE =1 表示使用代理 ，0表示不适用代理
PROXYUSE =  1
PROXYURL =  http://localhost:10809  

#币安密钥
APIKEY= 
SECRETKEY= 


**启动程序之后**
点击加载账户信息之后就可以实时监控数据，默认30s 刷新一次

**程序源码安装**
下载node 16版本
下载代码
进入目录
npm install
npm start
